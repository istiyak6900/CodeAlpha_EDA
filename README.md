# CodeAlpha - Exploratory Data Analysis

## Project Overview
This project is part of the Data Analytics Internship at CodeAlpha.

The project performs Exploratory Data Analysis (EDA) on the Netflix dataset using Python.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Features
- Dataset exploration
- Missing value analysis
- Movies vs TV Shows visualization
- Data understanding using charts

## Output
The project generates visual insights from Netflix content data.
